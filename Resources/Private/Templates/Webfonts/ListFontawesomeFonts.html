Fontawesome


Just add the following code to your typoscript. The extension will do the rest. This will install the `free` version of Fontawesome for the web. Make sure you website is eligible for using the free version. More information: https://fontawesome.com/plans

plugin.tx_webfonts.settings {
  fonts {
    20 {
      id=fontawesome
      provider=fontawesome
      version=6.4.0
      methods=css
      styles=all
      minified=true
    }
  }
}

Visit extension documentation for a detailed parameter description
